# Поисковик документов

## Функционал
Сервис позволяет по загруженному в папку `data` csv-файлу с определенным форматом(см.ниже) получать определенное количество текстов, содержащих это слово. Также можно удалять текст по его `id`

Для `.csv` необходимы колонки `text`,`created_date`(в формате `2006-01-02 15:04:05`, часовой пояс по умолчанию считается UTC+0), `rubrics`(в формате `"['VK-1603736028819866', 'VK-95883495386', 'VK-55347043459']"`). 

## Технологический стек

- Язык: Go 1.26.1
- СУБД: PostgreSQL 17+
- Elasticsearch: 8.x
- Сетевое взаимодействие: net/http
- Работа с Elasticsearch: `github.com/elastic/go-elasticsearch/v8`
- Работа с базой данных: `github.com/jackc/pgx/v5`
- Логирование: `log/slog`
- Агрегация ошибок: `go.uber.org/multierr`

## Запуск приложения

Для запуска приложения необходимо скачать Docker Desktop(https://www.docker.com/products/docker-desktop/)

После скачивания зайдите в корень проекта, создайте файл `.env`. Заполните его в соответствии с содержимым файла `.env.example`. В нем приведены необходимые переменные среды и их описание. Не забудьте про `DATA_PATH` - это и есть путь к `.csv` файлу в указанном ранее формате. При первом запуске приложение на основе этого файла заполнит базу данных и Elasticsearch.

Далее перейдите в консоль и введите следующие команды:
```bash
docker compose down -v
docker compose up --build
```

Приложение запущено и готово к работе. Подробнее о методах можно узнать в `docs.json` в корне проекта.
